Overview

Context

Guide is the primary control loop for the application. It defines four actions, checks for the restaurants text file (uses it if it can creates it if it is absent, or exits the app if it cant do either), and starts the loop so the user can list, find, add, or quit. Actions are class variables.

def launch!
  output_introduction
  # action loop
  result = nil
  until result == :quit
    action, args = get_action
    result = do_action(action, args)
  end
  output_conclusion
end
